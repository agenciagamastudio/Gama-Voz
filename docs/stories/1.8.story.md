# Story 1.8 — Sistema de Gamificação: Ações de Engajamento com Pontos Extras

**Epic:** GAMA_CALCULADORA
**Priority:** High
**Status:** Ready for Dev
**Created:** 2026-02-28
**Complexity:** Medium (3-5 pontos)

---

## 📖 Descrição

Implementar um sistema de **ações de engajamento** que incentiva usuários a interagir com a plataforma e ganharem pontos extras quando seus **pontos diários de energia (20 pts) se esgotarem**.

Cada ação completada resulta em 3-5 pontos adicionais, permitindo que usuários acumulem ~25-30 pontos extras através de diversas ações (compartilhamento, redes sociais, WhatsApp, etc.).

---

### Acceptance Criteria

- [ ] **AC3:** Pontos são concedidos atomicamente após ação completada (não antes)
- [ ] **AC4:** UI "Ganhe Pontos" aparece APENAS após usuário completar perfil

- [ ] **AC7:** Testes unitários para PointsContext com achievements (>80% coverage)
- [ ] **AC8:** Sem duplicação: não é possível ganhar pontos 2× pela mesma ação
- [ ] **AC9:** Bug dos "20 pontos repetidos" é resolvido (rechequear trigger)
