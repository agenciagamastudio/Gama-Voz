## 📝 Description

Migrar aplicação GAMA_VOZ_V2 para usar o novo Design System Gama v1.0.0 consolidado em `gama-ds-platform`. Esta story envolve atualizar cores, tipografia, espaçamento e componentes para seguir o novo padrão visual.

**Why:** Unificar visual de todos projetos Gama sob um Design System consistente, eliminar cores/tipografia antigas, garantir acessibilidade (WCAG AA).

**Impact:** Visual updates + Token standardization. Sem mudanças de lógica. Risk baixo.

## 🎨 Design System Reference

### Tokens Principais

**Cores:**
- Primary: `#88CE11` (verde neon Gama)
- Dark: `#161616` (background principal)
- Surface: `#272727` (cards/containers)
- Surface Accent: `#333333` (borders)
- Text: `#FFFFFF` (texto principal)
- Text Secondary: `#A1A1AA` (texto secundário)
- Success: `#10B981` | Warning: `#F59E0B` | Error: `#E11D48` | Info: `#3B82F6`

**Tipografia:**
- Heading: Poppins Bold (700-900)
- Body: Poppins Regular (400-500)
- Code: JetBrains Mono (400-600)

**Espaçamento:**
- xs: 4px | sm: 8px | md: 12px | lg: 16px | xl: 24px | 2xl: 32px | 3xl: 48px

### Exemplo de Aplicação (Tailwind)
```tsx
<button className="bg-gama-primary text-gama-dark font-black rounded-lg px-6 py-3 hover:brightness-110">
  Click me
</button>

<div className="bg-gama-surface border border-gama-surface-accent rounded-2xl p-6">
  Card content
</div>
```
