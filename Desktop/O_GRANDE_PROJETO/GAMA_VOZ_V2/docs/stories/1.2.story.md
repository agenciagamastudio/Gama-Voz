## 📝 Description

Refatorar GAMA_VOZ_V2 de Streamlit (Python) para **Next.js 14 + React + Tailwind CSS** para permitir integração fluida do Design System Gama v1.0.0 e melhor customização visual.

**Why:** Streamlit tem limitações severas com Design Systems. Next.js oferece controle total sobre UI, reutilização de componentes do Design System, e escalabilidade.

**Impact:** 
- ✅ Integração 100% do Design System Gama
- ✅ Componentes reutilizáveis com gama-ds-platform
- ✅ Customização visual sem limitações
- ✅ Mesma funcionalidade (Groq API para transcrição)
- ⚠️ Requer ~1 dia desenvolvimento (refactor, não nova feature)

## 🚀 Development Notes

### Para @dev:

1. **Start point:** `npm create next-app@latest gama-voz`
2. **Setup:**
   ```bash
   npm install tailwindcss @groq/sdk
   npx tailwindcss init -p
   ```
3. **Design System:**
   - Copiar cores de `DESIGN_SYSTEM_INTEGRATION.md`
   - Integrar em `tailwind.config.ts`
4. **Audio Capture:**
   - Usar `navigator.mediaDevices.getUserMedia()`
   - `MediaRecorder` API para captura
5. **Groq Integration:**
   - Mesmo endpoint que Streamlit
   - `whisper-large-v3-turbo` (mesma model)
   - Mesma lógica de transcrição
6. **History:**
   - localStorage para dev
   - File-based em produção (se needed)
