## Gama Studio

### Brand Identity
- **Name:** Gama Studio
- **Tagline:** "Construindo marcas de alto impacto"
- **Primary Color:** #88CE11 (Verde Neon)
- **Secondary Color:** #161616 (Dark)
- **Industry:** Marketing & Design Agency

### Logo Anatomy

```
┌─────────────────────────────┐
│                             │
│       G  E M A             │
│      ╱           ╲         │
│     ╱             ╲        │
│    ║    STUDIO    ║        │
│     ╲             ╱        │
│      ╲           ╱         │
│                             │
└─────────────────────────────┘

Structure:
• Letter "G" → Geometric, modernista
• Estilo: Sans-serif (Poppins Bold)
• Formato: Vertical com "STUDIO" abaixo
• Cor: Verde neon (#88CE11)
• Fundo: Transparente ou dark (#161616)
```

### Logo Variations

#### 1. Full Logo (Primary)
```

✅ Min width: 200px
✅ Padding: 20px around
```

#### 2. Icon Only (Secondary)
```
┌──────┐
│ [G]  │
└──────┘

✅ Use: Favicon, app icon, small spaces
✅ Min size: 64px × 64px
✅ Must maintain square aspect
```

#### 3. Horizontal (Tertiary)
```
┌────────────────────────────┐
│ [G]  GAMA STUDIO          │
└────────────────────────────┘

✅ Use: Headers, navigation bars
✅ Min width: 300px
```

#### 4. Stacked (For Narrow Spaces)
```
┌──────┐
│ [G]  │
│ GAMA │
│ STUDIO
└──────┘

✅ Use: Mobile sidebars, vertical layouts
✅ Min width: 120px
```

### Color Variants

#### Light Background (Recommended)
```
Background: #FFFFFF or light gray
Logo color: #88CE11 (Verde full color)
Text: #161616 (Black)
```

#### Dark Background
```
Background: #161616 or #0D0D0D
Logo color: #88CE11 (Verde full color)
Text: #FFFFFF (White)
```

#### Monochrome (B&W Only)
```
Logo color: #000000 (Black)
Text: #000000 (Black)

⚠️ Use ONLY when color not possible
```

### Clear Space

```
Minimum clear space around logo = 20px

┌─────────────────────────────┐
│ 20px │                20px  │
│      │  [LOGO]              │
│ 20px │                20px  │
└─────────────────────────────┘
```

### Size Guidelines

| Use Case | Min Size | Max Size | Notes |
|----------|----------|----------|-------|
| Website logo | 200px | 400px | Hero section |
| Favicon | 64px | 256px | Browser tab |
| Business card | 40px | 80px | Small but visible |
| T-shirt print | 300px | 500px | Quality matters |
| Billboard | 500px+ | Unlimited | Large format |

### What NOT to Do

```
❌ DO NOT: Rotate, skew, or distort
❌ DO NOT: Change aspect ratio
❌ DO NOT: Add drop shadows
❌ DO NOT: Outline with stroke
❌ DO NOT: Use different color scheme
❌ DO NOT: Add decorative elements
❌ DO NOT: Make it 3D
❌ DO NOT: Use gradient (if solid color defined)
```

## General Guidelines

### All Brands

#### Protected Space
```
╔════════════════════════╗
║  [min space] [LOGO] [min space]  ║
║                        ║
║  MINIMUM CLEAR SPACE:  ║
║  • Horizontal: Logo width/2
║  • Vertical: Logo height/2
╚════════════════════════╝
```

#### Minimum Sizes

| Format | Min Size | Reason |
|--------|----------|--------|
| Digital screen | 120px | Readability |
| Print (business card) | 1 inch | Industry standard |
| Favicon | 64px | Browser display |
| T-shirt print | 4 inches | Print quality |

#### File Formats

| Format | Use Case | Quality |
|--------|----------|---------|
| .svg | Web, scalable, recommended | Perfect scalability |
| .png | Fallback, social media | Good quality |
| .jpg | Compressed, low quality | Not recommended |
| .gif | Animated (rare) | Limited support |

### Color Accessibility

```
All logos tested for:
✅ Contrast ratio 4.5:1 minimum (WCAG AA)
✅ Color blindness simulation (deuteranopia, protanopia)
✅ Readability at small sizes (64px minimum)

If contrast fails:
→ Use outlined version
→ Add drop shadow for separation
→ Increase size for better visibility
```

### Responsive Behavior

#### Desktop (1920px+)
```
Logo size: 400-500px
Placement: Top left or centered
Padding: 40px from edges
```

#### Tablet (768-1024px)
```
Logo size: 200-300px
Placement: Top left or navigation
Padding: 20px from edges
```

#### Mobile (< 768px)
```
Logo size: 80-120px (icon only)
Placement: Navigation bar
Padding: 10px from edges
```

### Do's and Don'ts

#### ✅ DO

- Use provided SVG files
- Maintain aspect ratio
- Keep minimum clear space
- Use brand colors
- Scale proportionally
- Test on multiple devices
- Get approval before use

#### ❌ DON'T

- Rotate or skew
- Change colors
- Add effects/shadows
- Make it 3D
- Outline with stroke
- Compress/pixelate
- Use low-resolution versions
- Combine with other logos

### Usage Rights

```
GAMA Brands are owned by Grupo Gama

✅ Permitted:
• Internal use (company materials)
• Client materials (with permission)
• Marketing (on-brand)
• Merchandise (branded items)

❌ Not Permitted:
• Sell or license separately
• Use on competing products
• Register as trademark (elsewhere)
• Modify without approval
• Use at excessively small sizes
```

### Testing Checklist

Before using any logo:

- [ ] Clear space respected
- [ ] Minimum size requirement met
- [ ] Color contrast adequate (4.5:1+)
- [ ] Visible on intended background
- [ ] Approved by brand manager
- [ ] Tested on multiple devices
- [ ] File format appropriate
- [ ] SVG when possible
