## Voice & Tone

### Gama Studio

**Voice Personality:** Creative Strategist

**Core Characteristics:**
- Forward-thinking and innovative
- Collaborative and partner-focused
- Confident but approachable
- Passionate about brand impact

**Tone (Context-Dependent):**
- **Marketing materials:** Bold, inspiring, premium
- **Client communication:** Professional, strategic, enthusiastic
- **Social media:** Engaging, relatable, behind-the-scenes
- **Support:** Warm, helpful, expert

**Do's:**
✅ Use vivid, creative language
✅ Tell brand stories
✅ Reference strategy and impact
✅ Celebrate client wins
✅ Show personality

**Don'ts:**
❌ Sound generic or corporate
❌ Use jargon without explanation
❌ Be overly formal
❌ Ignore cultural context
❌ Make unsupported claims

**Example Messaging:**
```
"Your brand deserves more than visibility—it deserves impact.
We build identities that move markets."
```

## Visual Identity

### Color Psychology

**Gama Studio — #88CE11 (Verde Neon)**
- Psychology: Growth, innovation, energy, premium
- Usage: CTAs, highlights, brand accent
- Emotion: Excitement, forward-thinking
- Context: Best on dark backgrounds, high contrast

**Gama TV — #0066FF (Broadcast Blue)**
- Psychology: Trust, professionalism, authority, broadcast
- Usage: Primary identity, headers, trust elements
- Emotion: Confidence, reliability
- Context: Media, corporate, premium spaces

**Gama Rádio — #DC2626 (Warm Red)**
- Psychology: Energy, excitement, passion, warmth
- Usage: Energy accent, alerts, CTAs
- Emotion: Excitement, engagement
- Context: Audio, podcasts, dynamic content

**Gama Calendários — #8B5CF6 (Purple)**
- Psychology: Organization, wisdom, productivity
- Usage: Primary identity, controls, hierarchy
- Emotion: Calm focus, clarity
- Context: Productivity, planning, interfaces

### Typography Scale

**Primary Font: Poppins**
- **Headlines (4XL):** 48px, Weight 900 (Black)
- **Headlines (3XL):** 36px, Weight 800 (ExtraBold)
- **Subheadings (2XL):** 24px, Weight 700 (Bold)
- **Titles (XL):** 20px, Weight 600 (SemiBold)
- **Body (Base):** 16px, Weight 500 (Medium)
- **Labels (SM):** 14px, Weight 500 (Medium)
- **Captions (XS):** 12px, Weight 400 (Regular)

**Code Font: JetBrains Mono**
- **Inline code:** 14px, Weight 500
- **Code blocks:** 13px, Weight 400

### Spacing System

All spacing based on 4px grid:
- **XS:** 4px (micro-spacing)
- **SM:** 8px (small gaps)
- **MD:** 12px (default spacing)
- **LG:** 16px (component spacing)
- **XL:** 24px (section spacing)
- **2XL:** 32px (major spacing)
- **3XL:** 48px (layout spacing)
- **4XL:** 64px (section breaks)

### Border Radius

- **Sharp:** 0px
- **Small:** 4px
- **Medium:** 8px
- **Large:** 12px
- **XL:** 16px
- **Full:** 9999px (circles/pills)

**Guidelines:**
- Buttons: 8-12px (depends on button size)
- Cards: 12-16px
- Modals: 16px (premium feel)
- Inputs: 8px
- Avatars: 50% (full circle)

## Photography & Imagery

### Photography Style

**Gama Studio:**
- **Style:** Modern, vibrant, lifestyle
- **Color:** Saturated, true-to-brand colors
- **Composition:** Dynamic angles, movement
- **People:** Diverse, confident, aspirational
- **Context:** Brand work in progress, client success
- **Ratio:** 3:2, 16:9, square (1:1)

**Gama TV:**
- **Style:** High-energy, broadcast-ready
- **Color:** Professional, broadcast-accurate
- **Composition:** Wide shots, impactful visuals
- **People:** Professional, in-context
- **Context:** Studio, broadcast, corporate events
- **Ratio:** 16:9 (widescreen for TV)

**Gama Rádio:**
- **Style:** Studio, intimate, authentic
- **Color:** Warm, energetic, podcast-friendly
- **Composition:** Close-ups, personality-driven
- **People:** On-air talent, authentic
- **Context:** Studio moments, behind-the-scenes
- **Ratio:** Square (1:1 for social/podcast)

**Gama Calendários:**
- **Style:** Clean, minimal, organized
- **Color:** Cool, professional, calming
- **Composition:** Flat lay, typography focus
- **People:** Professionals organizing/planning
- **Context:** Workspace, productivity
- **Ratio:** 3:2, square (1:1)

### Illustration Style

**Consistent Across All Brands:**
- **Style:** Minimal, geometric, flat design
- **Color:** Brand-primary + supporting colors
- **Stroke:** 1.5-2px for consistency
- **Scale:** 64px minimum for visibility
- **Usage:** Icons, empty states, diagrams

**Icon Guidelines:**
- ✅ Use Material Symbols or Lucide React
- ✅ Maintain consistent stroke weight
- ✅ Ensure 1:1 aspect ratio (square)
- ✅ Test at 24px, 32px, 48px sizes
- ❌ Don't create custom icons unless necessary
- ❌ Don't mix icon libraries

### Stock Imagery Rules

**When Using Stock Photos:**
1. Priority: Original photography > Stock
2. If stock: Choose "premium" / "editorial" collections
3. Avoid: Generic, overused stock images
4. Diversity: Represent your audience authentically
5. License: Always verify commercial use rights

**Prohibited Imagery:**
❌ Outdated office stereotypes
❌ Overly filtered/artificial photos
❌ Unrelated placeholder images
❌ Low-quality/pixelated content
❌ Heavily watermarked images

## Animation Guidelines

### Motion Principles

**All brands use:**
- **Duration:** 300ms (normal), 150ms (fast), 500ms (slow)
- **Easing:** Ease-in-out (natural feeling)
- **Stagger:** 50ms between elements
- **Purpose:** Reduce cognitive load, guide attention

### Animation Types

#### 1. Micro-interactions (150ms)
```
Examples: Button hover, toggle switch, form validation
Easing: cubic-bezier(0.4, 0, 0.2, 1)
```

#### 2. Transitions (300ms)
```
Examples: Page navigation, modal open/close, state change
Easing: cubic-bezier(0.4, 0, 0.2, 1)
```

#### 3. Loading States (Infinite)
```
Examples: Spinners, skeleton screens, progress
Easing: cubic-bezier(0.4, 0, 0.2, 1)
Duration: 1.5-2s per cycle
```

#### 4. Entrance/Exit (500ms)
```
Examples: Page load, hero animations
Easing: cubic-bezier(0.34, 1.56, 0.64, 1) [Ease-out-back]
```

### Brand-Specific Animations

**Gama Studio:**
- Energetic, quick transitions
- Bounce effect on CTAs
- Smooth page scrolls
- Color transitions on hover

**Gama TV:**
- Broadcast-style transitions (fade/dissolve)
- Quick, punchy timing (150ms)
- Slide effects for content
- Fade animations on UI elements

**Gama Rádio:**
- Sound-wave effects (when possible)
- Pulsing animations for audio
- Quick, lively interactions
- Smooth fade-in on content

**Gama Calendários:**
- Smooth, predictable transitions
- Clear state changes
- Organized reveals (calendar grid, schedule)
- No distracting effects

## Accessibility Standards

### WCAG 2.1 Compliance (All Brands)

**Level AA Minimum** (Standard)
**Level AAA Target** (Premium)

### Color Contrast

| Element | Requirement | Example |
|---------|-------------|---------|
| Text on background | 4.5:1 (AA) | White on #88CE11 ✅ |
| Large text (18+px) | 3:1 (AA) | Any headline on brand color |
| UI components | 3:1 (AA) | Button border, focus indicator |
| Graphical objects | 3:1 (AA) | Icon, logo, brand element |

**Testing Tools:**
- WebAIM Contrast Checker
- WAVE Accessibility Tool
- Lighthouse (Chrome DevTools)
- axe DevTools

### Keyboard Navigation

✅ All interactive elements keyboard accessible (Tab key)
✅ Logical tab order (top-to-bottom, left-to-right)
✅ Visible focus indicators (minimum 3px, 3:1 contrast)
✅ No keyboard traps
✅ Skip links on pages with many links

### Screen Reader Support

✅ Semantic HTML (buttons, links, forms)
✅ Alt text on all images (descriptive, not "image of...")
✅ Form labels associated with inputs
✅ ARIA landmarks (nav, main, contentinfo)
✅ No automated voice instructions (users understand context)

### Motion & Vestibular

✅ `prefers-reduced-motion` media query honored
✅ No auto-playing animations > 5 seconds
✅ No content that flashes > 3 times/second (seizure risk)
✅ Optional: Parallax and complex animations

### Font & Readability

✅ Font size: minimum 14px for body text
✅ Line height: 1.5x for body, 1.2x for headings
✅ Letter spacing: adequate (no squished text)
✅ Line length: 50-75 characters max (readability)
✅ Avoid ALL CAPS for large text (harder to read)

## Writing Guidelines

### Tone & Voice Standards

#### Clarity
- ✅ Active voice (except for emphasis)
- ✅ Specific, concrete language
- ✅ Avoid jargon (explain if necessary)
- ✅ Short sentences (< 20 words)
- ✅ Use lists for > 3 items

#### Inclusivity
- ✅ Gender-neutral language
- ✅ Avoid ableist language
- ✅ Cultural sensitivity
- ✅ No exclusive terminology
- ✅ Relatable examples

#### Brand-Specific Examples

**Gama Studio:**
```
❌ "Optimize your branding infrastructure"
✅ "Create brands that people remember and love"

❌ "Utilize our strategic methodology"
✅ "We'll build your brand strategy together"
```

**Gama TV:**
```
❌ "Expand your media reach vectors"
✅ "Get your message on the screens your audience watches"

❌ "Broadcast dissemination protocol"
✅ "Your ads, prime time, maximum impact"
```

**Gama Rádio:**
```
❌ "Audio frequency optimization"
✅ "Make sure your listeners hear you loud and clear"

❌ "Acoustic properties enhancement"
✅ "Your voice on their favorite station"
```

**Gama Calendários:**
```
❌ "Temporal organization infrastructure"
✅ "Take control of your schedule"

❌ "Chronological management solution"
✅ "Never miss what matters"
```

### Capitalization & Punctuation

**Brand Names:**
- Always capitalize: "Gama Studio", "Gama TV", "Gama Rádio", "Gama Calendários"
- Group name: "Grupo Gama" (both capitalized)

**Headings:**
- Title Case (capitalize first & last, all nouns/verbs)
- Example: "Create Your First Brand Board"

**Product Features:**
- Capitalize feature names: "Brand Board", "Strategy Canvas"
- Lowercase generic: "dashboard", "button", "link"

**Punctuation:**
- Sentences end with periods (professional)
- Lists use periods if complete sentences
- Use em dashes (—) not hyphens for emphasis
- Avoid exclamation marks (except CTAs occasionally)

### Call-to-Action Language

**Strong CTAs:**
- Action verbs: "Create", "Build", "Launch", "Start", "Explore"
- Clear outcome: "Build Your Brand Identity" (not "Click Here")
- First person: "Create My Brand" (not "Create a Brand")
- Urgency (light): "Get Started", "Start Free", "Explore Now"

**Avoid:**
❌ Vague: "Submit", "Send", "Continue"
❌ Passive: "It will be created", "You can choose"
❌ Uncertain: "Try", "Maybe", "Possibly"

## Usage Scenarios

### Email Communication

**Gama Studio Emails:**
```
Subject: "Your brand story deserves the spotlight 🎬"

Body: Conversational, benefit-driven, link to portfolio
Footer: Logo (green), social links, premium feel
```

**Gama TV Emails:**
```
Subject: "See your brand on prime time"

Body: Data-driven, results-focused, broadcast terminology
Footer: Logo (blue), broadcast credentials
```

**Gama Rádio Emails:**
```
Subject: "Let's amplify your message"

Body: Energetic, personality-driven, audio example
Footer: Logo (red), podcast links
```

**Gama Calendários Emails:**
```
Subject: "Organize your week, multiply your impact"

Body: Clear, structured, time-saving focus
Footer: Logo (purple), productivity links
```

### Social Media

**Captions:**
- Hook in first line (grab attention immediately)
- Brand voice consistent
- Include call-to-action
- Hashtag strategy: 3-5 relevant tags
- Emoji usage: 1-2 max, brand-appropriate

**Posting Schedule:**
- Gama Studio: 3-4x/week (quality over frequency)
- Gama TV: 2-3x/week (data-driven posts)
- Gama Rádio: 4-5x/week (audience engagement)
- Gama Calendários: 2-3x/week (tips & updates)

**Content Mix:**
- 40% Educational (tips, tutorials, industry insights)
- 30% Promotional (features, offers, launches)
- 20% Community (user stories, testimonials, wins)
- 10% Behind-the-scenes (team, culture, authenticity)

### Presentations & Decks

**Slide Design:**
- ✅ One main idea per slide
- ✅ Use brand colors (primary + supporting)
- ✅ Plenty of white space
- ✅ Consistent typography
- ✅ High-quality imagery

**Structure:**
1. Title slide (brand, topic, date)
2. Agenda (what you'll cover)
3. Problem (why this matters)
4. Solution (your approach)
5. Results (proof of impact)
6. Call-to-action (next steps)

## Compliance Checklist

### Before Launch/Publishing

**Visual Identity:**
- [ ] Correct brand color used (#88CE11, #0066FF, #DC2626, or #8B5CF6)
- [ ] Typography matches guidelines (Poppins for headings, consistent sizes)
- [ ] Spacing follows 4px grid
- [ ] Logo used correctly (minimum size, clear space, not distorted)
- [ ] Imagery aligns with brand style
- [ ] No unauthorized brand modifications

**Accessibility:**
- [ ] Color contrast tested (minimum 4.5:1 for text)
- [ ] All images have alt text
- [ ] Forms have associated labels
- [ ] Keyboard navigation tested
- [ ] Screen reader tested (NVDA or JAWS)
- [ ] Focus indicators visible
- [ ] Motion respects `prefers-reduced-motion`

**Writing:**
- [ ] Brand voice consistent
- [ ] Tone appropriate for context
- [ ] Grammar/spelling checked
- [ ] No jargon without explanation
- [ ] CTAs clear and action-oriented
- [ ] Inclusive language used

**Approval:**
- [ ] Brand manager approved
- [ ] Legal reviewed (if needed)
- [ ] QA tested all interactions
- [ ] Mobile-responsive verified
- [ ] Cross-browser tested (Chrome, Firefox, Safari, Edge)
