## 🎯 Brand Personality

### Voice & Tone
**Personality:** Confident Financial Expert

**Core Traits:**
- Professional & trustworthy
- Knowledgeable & insightful
- Clear & transparent
- Forward-thinking
- Accessible (no jargon)

### Tone by Context
```
Market Analysis:  Data-driven, confident, analytical
Client Advice:    Clear, expert, reassuring
Marketing:        Aspirational, premium, empowering
Support:          Patient, helpful, thorough
Social Media:     Educational, engaging, thought-leading
```

### Example Messaging
```
"Seu patrimônio merece mais que gestão.
 Merece expertise de verdade."

"Invista com inteligência. Cresça com segurança."

"Tecnologia financeira feita para brasileiros
 que entendem o valor do tempo."
```

### Do's & Don'ts
✅ Do:
- Use data and numbers
- Be transparent about fees
- Show real client results
- Explain complex concepts simply
- Build trust through education

❌ Don't:
- Use "get rich quick" language
- Promise unrealistic returns
- Use crypto/meme language
- Be condescending
- Oversimplify complex topics

## 📸 Visual Style

### Photography
**Style:** Professional, aspirational, diverse
**Subject:** Professionals, families, meetings
**Color:** Warm tones, professional settings
**Mood:** Confident, successful, planning-focused

### Imagery Examples
- Business professionals in productive settings
- Diverse age/ethnicity representations
- Real-world financial planning scenarios
- Growth charts, upward trends
- Secure/protected concepts (safes, locks, shields)

### Avoid
- Generic stock photos
- Overly happy/fake expressions
- Stereotypical wealthy imagery
- Outdated office settings
- Unclear financial concepts

## 📐 Typography

**Primary Font:** Poppins
- Headlines: 700+ weight (bold, semibold)
- Body: 400-500 weight
- Numbers/Data: 600 weight (emphasis)

**Code Font:** JetBrains Mono
- For financial calculations
- For code/formulas

### Scale
```
H1 (Hero):     48px / 900 weight
H2 (Major):    36px / 800 weight
H3 (Section):  24px / 700 weight
H4 (Subsec):   20px / 600 weight
Body:          16px / 400-500 weight
Caption:       12px / 400 weight
```
